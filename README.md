# 🎨 AI × Creativity ポートフォリオ

このリポジトリは、**生成AI・音声対話・LLM効率化**など、  
「AI研究 × クリエイティブ技術」をテーマにした個人ポートフォリオです。  

既存のAI技術を組み合わせたり、自分で実装を試しながら、  
**創作支援や新しい表現の形を探る**研究・開発を行っています。  

👉 **公開サイト:** [https://penpe.github.io](https://penpen5tan.github.io)
